Introduction To Web Programming

INTRODUCTION TO CSC 121

The Course title CSC 121 which stands for Introduction To Web Programming

  • Web programming is the process of creating dynamic, interactive websites and web applications that are accessible through the internet. It involves using a combination of languages, frameworks, and technologies to build the various components of a website, including its structure, design, and functionality. The Course being a 100 Level second semester course. It carries a 3 credit unit load as such has a class duration of three hours per week.

  1. COURSE AIM AND OBJECTIVE

    2. AIM
    The primary aim of a web programming course is to equip students with the knowledge, skills, and practical experience necessary to design, develop, and deploy modern web applications. The course strives to build a comprehensive understanding of both client-side and server-side technologies, fostering the ability to create engaging, user-friendly, and functional web solutions.

OBJECTIVE

The objectives of this web programming course are designed to ensure that students not only acquire theoretical knowledge but also gain practical experience and skills essential for a successful career in web development. Through a combination of lectures, hands-on projects, and collaborative work, students will learn to design, build, and maintain robust web applications. The course will cover both fundamental and advanced topics, providing a comprehensive understanding of the web development ecosystem.
The specific objectives of the course are:

  1. Understand Web Technologies:
    • Gain a solid grounding in the core technologies of web development, including HTML, CSS, and JavaScript.
    • Learn how these technologies work together to create the structure, style, and interactivity of web pages.
  2. Develop Technical Proficiency:
    • Acquire hands-on experience with modern web development tools and frameworks.
    • Write clean, efficient, and well-documented code using best practices in web programming.
  3. Master Client-Side Development:
    • Build interactive and responsive web interfaces using HTML, CSS, and JavaScript.
    • Utilize libraries and frameworks such as jQuery, React, or Angular to enhance functionality and user experience.
  4. Explore Server-Side Development:
    • Understand the role of server-side scripting in web development.
    • Learn to use server-side languages such as PHP, Python, Ruby, or Node.js to handle data processing, manage user sessions, and interact with databases.
  5. Database Management:
    • Gain proficiency in designing and managing databases.
    • Learn to use SQL and NoSQL databases like MySQL, PostgreSQL, and MongoDB to store, retrieve, and manipulate data.
  6. Implement Web Security Practices:
    • Understand common security threats and vulnerabilities in web applications.
    • Learn to implement security best practices to protect against attacks such as SQL injection, cross-site scripting (XSS), and cross-site request forgery (CSRF).
  7. Deploy and Maintain Web Applications:
    • Learn the basics of deploying web applications to a web server.
    • Understand version control systems like Git for code management and collaboration.
    • Gain insights into maintaining and updating web applications post-deployment.
  8. Foster Problem-Solving and Analytical Skills:
    • Develop the ability to troubleshoot and debug web applications.
    • Enhance problem-solving skills by working on real-world projects and case studies.
  9. Promote User-Centered Design:
    • Emphasize the importance of user experience (UX) and accessibility in web design.
    • Learn to create web applications that are intuitive, inclusive, and usable for a diverse audience.
  10. Encourage Collaboration and Communication:
    • Develop teamwork and communication skills through group projects and peer reviews.
    • Learn to collaborate effectively with designers, developers, and other stakeholders in a web development project.

  • LEARNING OUTCOMES

    • Upon successful completion of this web programming course, students will be able to:

    • Design and Structure Web Pages
    • Develop Interactive User Interfaces
    • Implement Server-Side Functionality
    • Manage Databases
    • Ensure Web Security
    • Deploy and Maintain Web Applications
    • Solve Web Development Problems
    • Apply User-Centered Design Principles
    • Collaborate Effectively in Teams
    • Build a Portfolio of Web Projects
